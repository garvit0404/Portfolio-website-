Projects

Mediavibe Mediaplayer

Description of the project

Developed a fully functional media player website that allows users to browse and play songs by various artists. The platform includes user authentication and secure login functionality, leveraging MongoDB for storing user credentials.

Technologies Used:

HTML, Tailwind CSS, JavaScript, Node.js, Express.js, MongoDB

Key Features:

Play songs from a collection of various artists. Secure user authentication with MongoDB for storing user IDs and passwords. Server-side handling of authentication via Node.js and Express.js.

React-Weather Project

Description of the project

Built a weather forecasting application that provides real-time weather conditions for any location worldwide. Users can search for any city or country to view the current temperature, wind speed, and humidity, along with a 4-5 day forecast.

Technologies Used:

JavaScript, React.js, Tailwind CSS OpenWeatherMap API
(for Fetching Weather data) , Node JS

Key Features:

Search and display real-time weather data for any location globally. Provides extended weather forecasts (4-5 days). Utilizes the API for accurate weather information, including temperature, wind speed, and humidity.
